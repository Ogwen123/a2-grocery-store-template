How does that work?

When the page loads, it makes a request to the API. The API returns a JSON object with a key called "data". The value of that key is displayed on the page.

You can think of a JSON object as a Python dictionary

To see this in action, click F12 and open the network tab, then press F5.

(make sure you are not logged in to your Hwb account, try incognito)

The highlighted request is the one we are interested in!

If you can't find it, right click the "name" tab at the top, then enable "path" and look for a value called "/api/test"
